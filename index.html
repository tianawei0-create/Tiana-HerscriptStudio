About /
关于项目

Nüshu began as a hidden language between women. Here, it becomes a visible space of memory, voice, and connection.
女书曾是女性之间的隐秘语言，而在这里，它成为一个可被看见的记忆、声音与连结空间。

HerScript Studio is an independent creative project that brings together AI film, digital storytelling, cultural memory, interactive games, and a physical board-game extension. Centered on Nüshu, it transforms women’s hidden writing into a contemporary visual and interactive language.
HerScript Studio 是一个独立创作项目，融合了 AI 电影、数字叙事、文化记忆、互动游戏与实体桌游拓展。项目以女书为核心，将女性隐秘的书写传统转化为一种当代的视觉与互动语言。

Board Game Extension / 桌游拓展

NÜSHU MONOPOLY-STYLE BOARD GAME / 女书大富翁

This project extends into a physical board game that turns Nüshu culture into a playable journey. The board contains 41 connected cells, including Cultural Cards, Event Cards, Decision Cards, and digital website tasks. Players learn Nüshu symbols, encounter women’s life stories, make emotional choices, and build invisible links with others through gameplay.

本项目延伸为一套实体桌游，将女书文化转化为可玩的生命旅程。棋盘由 41 个连续格组成，包含文化卡、事件卡、决策卡与网站互动任务。玩家在游戏中开始学习符号，进入女性生命故事，做出情感选择，并与他人建立“看不见的连结”。
